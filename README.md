# Pong Game

## Description
Pong is a classic arcade game that simulates table tennis. The game features two paddles and a ball. The objective is to score points by hitting the ball past the opponent's paddle.

## How to Play
1. Open the game in your web browser.
2. Use your mouse to control the paddle:
    - Move the mouse up to move the paddle up.
    - Move the mouse down to move the paddle down.
3. The game starts when the ball is served from the center of the screen.
4. Hit the ball with your paddle to send it towards the CPU opponent.
5. Score a point when the ball passes the CPU opponent's paddle.
6. The game continues until a player reaches the winning score.

Enjoy playing Pong against the CPU in your browser!
